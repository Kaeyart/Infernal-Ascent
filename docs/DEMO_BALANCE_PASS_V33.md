# V33 — Demo Balance Pass

## Goal

Make the current demo run fairer, clearer, and closer to the target 10–20 minute run without adding content or changing systems.

This pass is a tuning pass only.

## What this pass changes

### Player

- Max HP raised from 6 to 7 for a more forgiving demo baseline.
- Contact-damage i-frames increased to reduce chain-hit frustration.
- Light/heavy attack arcs widened slightly to reduce missed attacks caused by isometric direction ambiguity.
- Dash cooldown slightly reduced.
- Attack movement commitment loosened slightly so combat feels less sticky.

### Enemies

- Early enemy scaling starts later.
- Ash Grunt windup/cooldown made clearer.
- Cinder Lunger lunge slowed and windup increased.
- Ember Spitter projectile slowed and attack rhythm made more readable.
- Chainbound Penitent made less spongy but still heavy.
- Furnace Imp made less obnoxiously fast.
- Bell Wretch support pulse weakened and slowed.

### Hazards

- Hazard windup increased.
- Active time tightened slightly.
- Cooldown increased slightly.
- Knockback reduced.

### Ash Warden

- Boss health lowered to 90 for demo pacing.
- Boss attack windups made slightly more readable.
- Furnace Seal Stagger reward increased.
- Summons reduced so the boss fight stays about the boss, not enemy clutter.

### Run economy

- Victory grants 4 Ash Sigils.
- Death grants 1 base Ash Sigil, so failed runs still progress.
- Fountain heals 65% max HP.

## What this pass does not change

- No new rooms.
- No new enemies.
- No new rewards.
- No new boss mechanics.
- No save schema changes.
- No UI redesign.
- No player art changes.
- No run-flow changes.

## Definition of done

- Normal player can clear early rooms.
- Boss kills careless player.
- Good player can win first or second try.
- Rewards feel useful.
- Healing matters.
- Death does not feel random.
- Full run trends toward 10–20 minutes.
