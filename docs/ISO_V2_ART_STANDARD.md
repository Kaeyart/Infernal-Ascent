# Infernal Ascent Iso V2 Art Standard

## Projection

Use true isometric 2D.

```text
Floor tile: 64x32 diamond
Grid axes: diagonal screen axes
Vertical height increments: 32px or 64px
```

Do not mix the old top-down square floor tiles with isometric wall facades.

## Room shell logic

Each room needs a real isometric shell:

```text
north/rear wall
east wall
west wall
front/south edge
corner pieces
floor diamonds inside the shell
foreground framing
```

## Styling

```text
black basalt
ash gray stone
dark iron
muted brass
controlled ember orange
muted infernal red
```

Avoid random lava spam, boss-room throne energy, and overdecorated props in early rooms.
